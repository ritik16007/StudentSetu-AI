# StudentSetu AI

StudentSetu AI is a student productivity web app that helps students manage important academic documents, deadlines, formal emails, and loan or scholarship applications in one simple dashboard.

## Features

- Student document manager
- Deadline tracker
- Mail draft generator
- Loan and scholarship application tracker
- Search and filter records
- Status update option
- Dashboard live counts
- Print summary
- Export data as JSON
- Import JSON backup
- LocalStorage support

## Tech Stack

- HTML
- CSS
- JavaScript
- LocalStorage

## Purpose

This project is built to solve real problems faced by students during college admission, scholarship application, education loan process, and document management.

## How to Use

1. Open `index.html` in your browser.
2. Add your documents, deadlines, and applications.
3. Use the search and filter option to find records.
4. Generate formal emails using the mail draft generator.
5. Export your data as JSON for backup.
6. Import JSON backup whenever needed.
